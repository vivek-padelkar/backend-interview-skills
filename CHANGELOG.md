# Changelog

All notable changes to the Backend Interview skill will be documented in this file.

## [1.0.0] - 2026-05-22

### Added
- Initial release of Backend Interview skill
- Full mock interview session with 5-8 scenario-based questions
- Support for 4 experience levels: Junior, Mid, Senior, Staff
- 8 topic areas: Node.js Core, Express/Fastify, Databases, System Design, Microservices, Security, Performance, Testing
- 3 difficulty levels: Easy, Medium, Hard
- 5 company interview styles: Amazon, Google, Meta, Netflix, General MANG
- Real-time feedback system with grades (✅ Strong, 🟡 Partial, ❌ Needs Work)
- Comprehensive end-of-session analysis report
- Curated learning resources (YouTube, LinkedIn, articles, GitHub repos)
- Question bank with 200+ scenarios across all difficulty/topic/experience combinations
- Company-specific question re-phrasings and expectations
- Study plan generation based on identified gaps

### Features
- Dynamic question generation based on user selections
- Real-time feedback on each answer
- Score breakdown per question (out of 10)
- Overall pass rate and final percentage score
- Strength identification and confidence builders
- Top 3 priority study concepts
- Curated prep materials for each topic
